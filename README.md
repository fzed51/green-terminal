# GreenTerminal UI

React
TypeScript  
License: MIT

**GreenTerminal UI** est une bibliothèque de composants React au thème *"dark hack green"*, inspirée des interfaces rétro des terminaux informatiques. Parfaite pour les dashboards techniques, les applications cyberpunk ou tout projet nécessitant une ambiance immersive et futuriste.

---

## 📦 Installation


### Prérequis

- React >19

### Commande

Installe le package via npm ou yarn :

```bash
npm install @fzed51/green-terminal
# ou
yarn add @fzed51/green-terminal
```

## 🛠 Développement

### Prérequis

- Node.js (v22+)
- yarn

### Lancer le projet en local

1. Clone ce dépôt.
2. Installe les dépendances :
  ```bash
   npm install
  ```
3. Lance le serveur de développement :
  ```bash
   npm run dev
  ```
